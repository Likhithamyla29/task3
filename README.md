# E-Commerce Sales Data Analysis Using SQL

## Project Overview

This project focuses on analyzing an E-Commerce Sales dataset using SQL queries. The objective of the task was to import the dataset into a database table, perform data analysis using SQL, and extract meaningful business insights from the sales data.

## Dataset Information

The dataset contains information related to customer purchases, product categories, sales revenue, shipping details, and order information.

Columns used in the analysis:

* Customer_ID
* Gender
* Region
* Age
* Product_Name
* Category
* Unit_Price
* Quantity
* Total_Price
* Shipping_Fee
* Shipping_Status
* Order_Date

## Steps Performed

### 1. Dataset Preparation

* Downloaded the E-Commerce Sales dataset.
* Reviewed the dataset structure and column names.
* Verified data consistency and column formats.

### 2. Database Setup

* Created a table named `ecommerce`.
* Imported the dataset into the database table.
* Verified that all records were successfully loaded.

### 3. SQL Analysis

The following SQL queries were executed to analyze the dataset:

1. Retrieved the top 5 products with the highest total sales.
2. Calculated total revenue generated by each category.
3. Counted the number of orders placed in each region.
4. Computed the average customer age by gender.
5. Identified the top 10 customers based on spending.
6. Calculated the total quantity sold for each category.
7. Determined the average shipping fee for each shipping status.
8. Generated month-wise revenue analysis.
9. Listed products with sales greater than 10,000.
10. Identified the region generating the highest revenue.

### 4. Result Validation

* Executed all SQL queries successfully.
* Verified query outputs.
* Documented the results for reporting purposes.

## Key Insights

* Laptops generated the highest sales revenue among all products.
* Electronics contributed the highest revenue among categories.
* The East region generated the highest overall revenue.
* Customer spending patterns were analyzed using aggregate functions.
* Monthly revenue trends were identified through date-based analysis.

## Tools Used

* MYSQL
* Online MYSQL Compiler
* Microsoft Excel


This project demonstrates the use of SQL for data analysis by applying filtering, aggregation, grouping, sorting, and reporting techniques on an E-Commerce Sales dataset. The analysis provides useful business insights regarding customer behavior, sales performance, product demand, and regional revenue generation.
